# Find the Missing Letters

A lightweight browser game built with plain HTML/CSS/JS. Guess the hidden letters to complete the word. Keyboard and mouse supported.

## How to run
Open `index.html` in your browser. No build step required.

## Gameplay
- A word is shown with several letters hidden.
- Click letter keys or type on your keyboard to guess the missing letters.
- Correct guesses reveal letters and add score.
- Wrong guesses reduce lives. When lives reach 0, the game ends.
- Press Enter (or click New Round) to start the next round. Reset restarts the game.

## Difficulty
- Easy: fewer letters hidden.
- Medium: balanced.
- Hard: more letters hidden.

## Tech
- No dependencies. Works offline.
- Accessible labels and ARIA live regions for feedback.
